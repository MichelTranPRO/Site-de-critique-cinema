# CineCritique 2025

Notre projet **"CineCritique"**, un site web dynamique de critique de cinéma. Ce projet a été réalisé dans le cadre de la **SAÉ 2.3** (Installation de services réseaux / Développement Web) de l'IUT de Sénart-Fontainebleau en 2024.

## Description du projet

**CineCritique** est une plateforme permettant aux utilisateurs de consulter des informations sur des films, de lire des critiques et de partager leurs propres avis. Le site propose une interface intuitive pour naviguer dans un catalogue cinématographique, gérer des comptes utilisateurs et interagir avec une base de données de films.

Le projet a été développé en utilisant :

- **CodeIgniter** 
    
- **PHP** 
    
- **SQL (MariaDB)**
    
- **HTML5 / CSS3**
